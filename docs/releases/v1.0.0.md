# v1.0.0 - Initial open source release

## Highlights

- First public release of Gmail Alias Generator
- Static frontend app ready for Cloudflare Pages
- Bulk alias generation with dot / plus / case / domain variations
- Pagination support for large result sets
- Added README, LICENSE, CONTRIBUTING, SECURITY, CI, and community templates

## Verification

- `npm run check`
- GitHub Actions CI workflow included

## Notes

- This project is not affiliated with Google or Gmail
- Case/domain variants should be treated as delivery-equivalent forms where applicable
